Detailed Description

Implements a shared interface for interfacing with flight controllers. Application usually exports fc_sitl_client_interface to the flight controller and then flight controller polls the SITL for new data as well as sending back control inputs to the sitl.Since each flight controller is running in a separate thread and sitl application is reading the same data that flight controller is writing, this interface also provides means to synchronize data access between sitl and the flight controller. For the most part pthread mutexes are enough for this.
